# Changelog

Todas as mudanças relevantes deste projeto serão documentadas aqui.

O formato segue uma adaptação de [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/)
e as mensagens de commit seguem [Conventional Commits](https://www.conventionalcommits.org/).

## [0.1.0] - 2026-06-26

### Adicionado

- Aplicação desktop em Python com CustomTkinter.
- Pipeline modular para transcrição em turco com faster-whisper.
- Tradução automática para português do Brasil.
- Geração de legendas SRT sincronizadas.
- Processamento em fila, cancelamento, histórico, pré-visualização e logs.
- Configuração persistente em `config.json`.
- Suporte a CUDA quando disponível e fallback para CPU.
- Estrutura de testes, documentação, PyInstaller e GitHub Actions.
